## List API
- Login, Register, ChangePassword, 
- Update Profile
- Comment
- Favorite Movie

## Installation
- Clone the Project
```bash
npm install
```
- Create file .env
```bash
CONNECT_MONGODB_ATLAT_URL= Link mongoDBAtlat to connect your database
```
- Run Project
```bash
npm start
```
